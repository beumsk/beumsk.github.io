Markdown is a simple but widely used language, especially for documentation.

Every section has the syntax first and the rendered example right after.

Let's see how we can write with it!

## Text

```
**bold text**
*italicized text*
~~strikethrough text~~
```

**bold text**, _italicized text_ and ~~strikethrough text~~

## Heading

```
# H1
## H2
```

# H1

## H2

## Blockquote

```
> blockquote
```

> blockquote

## Ordered list

```
1. First item
2. Second item
3. Third item
```

1. First item
2. Second item
3. Third item

## Unordered list

```
- First item
- Second item
- Third item
```

- First item
- Second item
- Third item

## Code

```
`code`
```

`code`

## Block code

````
```
// some line of code
```
````

```
// some line of code
```

## Horizontal rule

```
---
```

---

## Link

```
[title](https://remybeumier.be)
```

[title](https://remybeumier.be)

## Image

```
![alt text](/images/rb.jpg)
```

![alt text](/images/rb.jpg)

## Task list

```
- [x] Completed
- [ ] Uncompleted
```

- [x] Completed
- [ ] Uncompleted

## Table

```
| First col title | Second col title |
| ----------- | ----------- |
| row1 col1 | row1 col2 |
| row2 col1 | row2 col2 |
```

| First col title | Second col title |
| --------------- | ---------------- |
| row1 col1       | row1 col2        |
| row2 col1       | row2 col2        |

As you can see it's not difficult and it's rather easy to remember.
